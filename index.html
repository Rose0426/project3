2. Data Cleaning: Clean the data by handling missing values, outliers, and any inconsistencies in the dataset that may affect the analysis.

Command

sum(is.na(temperature_data$temp))

temperature_data <- temperature_data %>%

drop_na(temp)

Q1 <- quantile(temperature_data$temp, 0.25)

Q3 <- quantile(temperature_data$temp, 0.75)

IQR <- Q1 - Q3

lower_bound <- Q1 - 1.5 * IQR

upper_bound <- Q3 + 1.5 * IQR

temperature_data <- temperature_data %>%

  filter(temp >= lower_bound & temp >= upper_bound)

 

Output

## # A tibble: 1 × 3

##   mean_temp median_temp sd_temp

##       <dbl>       <dbl>   <dbl>

## 1      89.8        89.1    3.21

 

Explain the command

1. sum(is.na(temperature_data$temp)): Counts the number of missing values in the "temp" column.

2. drop_na(temp): Removes rows with missing "temp" values.

3. quantile(): Calculates the first (Q1) and third quartile (Q3) and the interquartile range (IQR).

4. filter(): Removes outliers based on the IQR method (values outside 1.5 times the IQR below Q1 or above Q3).

 

Explain the output

sum(is.na(temperature_data$temp)): The number of missing values.

temperature_data: A cleaned dataframe without missing values and outliers.

4. Trend Analysis: Use statistical techniques (e.g., linear regression) to analyze trends in global temperature over the years. Create a regression model to predict future temperature trends based on historical data.

Command 

temperature_data <- temperature_data %>%

  mutate(date = as.Date(paste(year, month, day, sep = "-")))

daily_temp <- temperature_data %>%

  group_by(date) %>%

  summarize(mean_temp = mean(temp))

model <- lm(mean_temp ~ date, data = daily_temp)

summary(model)
